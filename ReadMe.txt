Multi-Layer Random Forest [for Image] Segmentation

Generate dataset of synthetic images/labels:
    mlrfsSynthData.m

Train with:
    mlrfsTrain.m

Segmentation example:
    mlrfsSegment.m

For a running example, run the above .m scripts in order.
See m-files for parameters to set, and their meaning.

----------

Reference:
	Becker C, Ali K, Knott G, Fua P.
	Learning context cues for synapse segmentation.
	IEEE Trans Med Imaging. 2013 Oct;32(10):1864-77.
